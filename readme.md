
# booknotes
    1、以MarkDown形式组织的读书笔记，学习笔记，建议用Typora编辑器阅读；
    2、读者注意，该笔记并非按照书本知识点，而是按照个人的阅读和知识体系组织，有些简单或者已经掌握的会省略，力求详略得到;
    3、当书本的知识点我暂时没有理解时，将遵循原有书本的表述。
